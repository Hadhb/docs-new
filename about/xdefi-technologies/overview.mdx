# Overview of XDEFI Technologies

XDEFI Technologies provide tools to manage and interact with blockchains more efficiently, while letting users having full control and ownership over their assets.

## XDEFI Wallet
XDEFI Wallet is a multichain Wallet that allow users to interact with the [supported blockchains](./supported-blockchains.md), view their assets balances, NFT's and access dApps.
